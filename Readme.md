# eslint-config-vdemedes

My personal customizations to default [xo](https://github.com/sindresorhus/xo) rules.


### Installation

```
$ npm install eslint-config-vdemedes --save-dev
```


### Usage

in **package.json**:

```
"xo": {
	"extends": "eslint-config-vdemedes"
}
```
